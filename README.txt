VERSIÓN HTML SIN XAMPP

1. Abre una terminal dentro de esta carpeta.
2. Ejecuta: python -m http.server 8080
3. Abre http://localhost:8080

También puedes abrir index.html directamente y cargar el Excel con el botón, aunque algunos navegadores bloquean la carga automática del archivo inicial.

Los datos se guardan en el almacenamiento local del navegador de ese equipo. Esta versión no comparte información entre computadoras. Al cargar otro Excel, actualiza coincidencias y agrega registros nuevos.
